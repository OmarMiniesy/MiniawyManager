### Flags and Functions
> print table, with rows and columns    `-T`

> sort table, by column                 `-S <column>`

> search table by name/pid              `-F <name/pid>`

> filter table by column                `-C <column> <value>`

> filter and sort table                 `-C <column> <value> -S <column>`
